How to run :-

1 ) Run test_prg ( This is the server Game )
2 ) Run ./server ( This is the server Daemon )
3 ) Run ./client ( This is the client Daemon )   -- Please allow 20 sec before running ./test_prg_client so that server will transfer map to client daemon 
4 ) Run ./test_prg_client <argument>    (This is a Client Game)
